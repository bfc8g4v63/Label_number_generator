【Label number generator_VBA v1.0.4】

執行方式：
1. Label number generator_VBA.xlsm (巨集EXCEL)

功能：
1.輸入起始序號。
2.輸入結束序號。
3.輸入每箱有多少個序號。
4.箱號內建C/NO.，請輸入1 或 其它格式。
5.操作產生器產生於第一個分頁。
6.標籤機軟體匯入第一個分頁內容。

版本：
修正長度單格上限計算錯誤
修正標籤機軟體讀取錯誤
修正函數類性
修正溢位錯誤

環境：
僅支援office excel 2019+
＊開啟巨集方法＊
1.EXCEL>檔案>選項>信任中心>巨集設定>啟用VBA巨集
2.EXCEL>檔案>選項>信任中心>信任位置>新增位置(指定信讓的路徑文件夾[父層以下])
3.如開啟來自網路磁碟(SMB)，則需開啟EXCEL>檔案>選項>信任中心>信任的文件>允許信任特定網路上的文件，再不行則下載Tool網頁中的
allow_macro_from_internet.reg.reg>雙擊開啟解鎖 (如無法請使用 以系統管理員執行)。
